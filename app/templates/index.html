Welcome to The Bus Ta Move-ment!

This is the official fan page of Bus Ta Move. The go-to website for information about the critically acclaimed University of Michigan traveling website.

Problem that Bus Ta Move solves

The lack of knowledge that University Of Michigan students have when it comes to getting around campus efficiently

This problem is shared by all students regardless of class standing. This website attempts to solve the problem of:

the naive freshman who's new to Campus.

The Sophmore who only had Central Campus Classes last year but now has classes on North Campus.

The Juniors and Seniors who still have campus blindspots and have to go to their class discussion but have no idea where Lane Hall is

Bus Ta Move's Solution

The Bus Ta Move Team, consisting of Hussam Hashem, Junius Murphy, Wesley Pederson, and Kevin Wang decided to create a website in which the user can either input start and end on-campus locations , input their class schedule, or choose the option for the website to randomly generate start and end on-campus locations. From there, the user is given a map of the University's campus where they can travel and explore the campus right on their computer. The user has the option of walking around campus by foot or by riding the bus system. Each decision the user makes decreases the time-clock that exists in the game. With the map and decision-sensitive time-clock at hand, the user has a convinient and powerful tool to preplan their routes to class and other event's on-campus to ensure a timely arrival
