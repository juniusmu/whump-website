Problem that Bus Ta Move solves

A lot of University of Michigan students do not know how to effeciently travel around campus. This problem is shared by all students regardless of class standing. Bus Ta Move solves this problem. Bus Ta Move is for the naive freshman who are new to campus, the Sophmores who only had Central Campus Classes last year but now have classes on North Campus, and the Juniors and Seniors who have to go to their class discussion but have no idea where Lane Hall is.

Bus Ta Move's Solution

The Bus Ta Move Team, consisting of Hussam Hashem, Junius Murphy, Wesley Pederson, and Kevin Wang decided to use the reactjs framework to create a website in which the user can either input start and end on-campus locations , input their class schedule, or choose the option for the website to randomly generate start and end on-campus locations. From there, the user is given a map of the University's campus where they can travel and explore the campus right on their computer. The user has the option of walking around campus by foot or by riding the bus system. Each decision the user makes decreases the time-clock that exists in the game. With the map and decision-sensitive time-clock at hand, the user has a convinient and powerful tool to preplan their routes to class and other event's on-campus to ensure a timely arrival.
